# Blinkit Grocery Sales Dashboard

## 📊 Project Overview

An end-to-end Power BI sales analytics dashboard developed to analyze Blinkit's grocery sales performance, customer behavior, outlet performance, and product category trends. The dashboard enables stakeholders to monitor key business metrics and make data-driven decisions.

## 🎯 Business Objective

To provide actionable insights into sales performance, customer ratings, product demand, and outlet efficiency through interactive visualizations and KPI tracking.

## 📈 Key Metrics

* Total Revenue: $1.20M+
* Average Sales
* Total Item Count
* Customer Ratings
* Outlet Performance Analysis
